Syntax


The Syntax of SU# Is simple it is Case-Sensitive or in
Simple terms it must be exactly
as specified,

Example:
@inu:input:Where do you want to go?:google.com:https

Dont worry if it looks weird,
Complicated or more as it will
Make sense later!
As shown you must follow everything with a :,
exept from the last argument which in this
case is https,

Page 4
Current functions and more.


The current functions are:
var,
var takes 2 arguments | the name and value,
Example:

var:oba:10
This will make a variable named oba and it will have a value of 10.

write,
var takes 1 argument | value,
Example:

write:hi
This will output hi,
You can use variables you have made
with the name surrounded in 's like this:
'oba' and it will write it out or you can
just use the variable like this:
write:oba. note: you can use a variable in write, alt, log @inu, @in and ntw,

alt,
alt takes 1 argument | the value,
Example:

alt:hi
This will alert the text hi,

log,
var takes 1 argument | the value,
Example:

log:This is a test log!
This will add "This is a test log" to
the log array,

@in,
@in takes 3 arguments | the output name, title, and preset value,
Example:

@in:out:Are you a doge?:no
This will create a varible with the input from the
input named "Are you a doge?", and a default input value
of "no".

@inu,
@inu takes 4 arguments | the output name, title, preset value and the protocol
Example:

@inu:out:Enter your website?:google.com:https
This will create a varible with the input as a url from the
input named "Whats your website?", and a default input value
of "google.com" with its protocol as "https".

ntw,
var takes 1 argument | the url,
Example:

ntw:google.com
This will open a google.com in a newtab.
